# 福建十三水

本项目是由纯python编写的福建十三水游戏。

## 运行环境

操作系统： windows或者Linux ; python 3 ；模块依赖参照requirements.txt

```bash
pip install -r requirements.txt
```

## 编译与运行

安装完依赖可直接运行根目录下LoginWin.py

```
python LoginWin.py
```
